# AI Usage in This Website Project

## Purpose
This document explains how AI was used during the creation of this website project.

## How AI Was Used
AI was used as a coding assistant to support planning, implementation, and refinement of the site.

### 1. Project Setup and Structure
- Helped create the initial project structure.
- Suggested a clean separation between HTML content and CSS styling.
- Supported organizing files into logical folders (for example, images).

### 2. HTML and Content Editing
- Assisted in building and refining the page sections.
- Helped update text content and move content between sections when needed.
- Helped improve headings, form labels, and overall readability.

### 3. Styling and Responsiveness
- Suggested CSS improvements for layout and readability.
- Added mobile-friendly improvements (for example, responsive image behavior).
- Helped keep the visual structure consistent across screen sizes.

### 4. Form and Delivery Flow
- Assisted in connecting the contact form to an external form service.
- Helped configure a submission flow with a thank-you page.
- Helped improve validation and input limits for cleaner form data.

### 5. Security Improvements
- Helped add practical security hardening for a static site, including:
  - Content Security Policy (CSP)
  - Referrer policy
  - Anti-spam honeypot field
  - Input constraints and safer form behavior

### 6. Publishing and Version Control Support
- Assisted with Git workflows (commit and push steps).
- Helped with GitHub Pages configuration and troubleshooting.

## Human Decisions and Responsibility
The final decisions, content choices, and project direction were made by the project owner. AI support was used to speed up development and improve quality, but the owner reviewed and approved the results.

## Summary
AI was used as a practical development assistant for implementation, editing, security hardening, and publishing guidance throughout this website project.
