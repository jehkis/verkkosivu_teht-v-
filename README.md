# WS01 HTML Practice Project

This project implements all tasks from Workshop 1 (Introduction to HTML).

## Project Structure

- `index.html`: Main page with all WS01 tasks
- `styles.css`: Basic styling for readability
- `images/profile-placeholder.svg`: Placeholder image used in Task 1

## Implemented Tasks

1. Basic HTML page with title, heading, paragraph, and image.
2. Ordered and unordered hobby lists.
3. Hyperlink to a favorite website.
4. Course table (3 rows + headers) and contact form.
5. Ready for HTML validation with W3C Validator in VS Code.

## How to Open

1. Open `index.html` in your browser.
2. Or use VS Code Live Server extension if you prefer local serving.

## Validation (Task 5)

1. Install **W3C Web Validator** extension in VS Code.
2. Open `index.html`.
3. Run validation from the extension.
4. Fix reported errors or warnings.

## Learning Path (Step-by-Step)

1. Start with `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>`.
2. Add meaningful content blocks with headings and paragraphs.
3. Add media (`<img>`) with descriptive `alt` text for accessibility.
4. Add lists with `<ol>` and `<ul>` depending on meaning.
5. Add links with `<a href="...">`.
6. Add tabular data using `<table>`, `<thead>`, and `<tbody>`.
7. Add form controls with `<label>` + `<input>` and required fields.
8. Separate structure (HTML) from appearance (CSS) in `styles.css`.
9. Validate and iterate.
