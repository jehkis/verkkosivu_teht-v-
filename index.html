01 / Esittely

Mitä tämä sivu kertoo minusta

Moikka, olen Joona. Opiskelen tietojenkäsittelyä Laurean ammattikorkeakoulussa ja työskentelen opintojen ohessa ohjaajana Rinnekodeilla. Vapaa-aika menee usein urheillessa, ulkona liikkuessa ja perheen kanssa.

03 / Workshopit

WS01, WS02, WS03 ja WS04 yhdellä silmäyksellä

WS01 HTML-perusta

Ensimmäinen versio, jossa rakenne, kuva, lista, taulukko ja lomake ovat kasassa.

 WS02 CSS Fundamentals

Työpaja, jossa sivulle rakennettiin oma perusilme, värit, spacing ja lomake-tyylit.

 WS03 Advanced CSS

Hieman villimpi versio, jossa mukaan tuli animaatioita, hoverit, filtterit ja pseudo-elementit.

 WS04 Page Layout Techniques

Semanttinen rakenne, grid, flexbox ja media queryt samalla sivulla harjoiteltuna.
